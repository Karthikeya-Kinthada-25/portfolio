# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **React + Vite + Tailwind CSS** to showcase my projects, certifications, and achievements.

---

## 🚀 Features

- ✨ Clean and modern UI
- 📱 Fully responsive design
- 📂 Projects showcase section
- 🎓 Certifications display
- 📄 Documents & resume section
- 🎥 Video section for demos
- ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Version Control:** Git & GitHub
